Data columns description, left to right:

1. Time, s;
2. Sound velocity (ISA), m/s;
3. Air density (ISA), kg/m^3;
4. Static air pressure (ISA), Pa;
5. Static temperature (ISA), K;
6. Climb mode (0: Climb, 1: Descend, 2: Level, 3: Error);
7. Flight phase (0: Upper Descent, 1: Lower Descent, 2: Approach, 3: Landing, 4: Error);
8. Reduced power mode (0: On, 1: Off, 2: Error);
9. Speed hold mode (0: CAS, 1: Mach, 2: Error);
10. Troposphere mode (0: Low, 1: High, 2: Error);
11. Acceleration mode (0: Accelerate, 1: Decelerate, 2: Constant, 3: Error);
12. Target altitude, ft;
13. Target speed TAS, kt;
14. Cross track error, nm;
15. Heading error, deg;
16. Longitude, deg;
17. Latitude, deg;
18. Altitude, ft;
19. True airspeed, kt;
20. True heading, deg;
21. Mach number;
22. Bank angle, deg;
23. Path angle, deg;
24. Energy share factor;
25. Lift coefficient;
26. Drag coefficient;
27. Lift force, N;
28. Drag force, N;
29. Thrust force, N;
30. Navigation mode;
31. Cleared for final approach flag;
32. On final approach flag (localizer captured);
33. On final path flag (glideslope captured);
